# 現在使用中のスクリプト

## 自動実行（GitHub Actions）
- run-daily-reviews-v3.cjs - 毎日の口コミ自動投稿
- fetch-keiba-sites.js - サイト自動検知
- check-new-reviews.cjs - 新規口コミチェック

- manage-site-quality.cjs - 悪質サイト管理
- upload-adjusted-reviews.cjs - 口コミアップロード
- puppeteer-screenshots.cjs - スクリーンショット取得
- optimize-screenshots.cjs - 画像最適化
